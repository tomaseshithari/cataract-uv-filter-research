# Cataract UV Filter Research

This collaborative research project aims to analyze the long-term efficacy of various UV-filter compounds in preventing cataract formation. We will review existing data on UV filter mechanisms, epidemiological correlations, and synthesize recommendations for optimal ocular protection.

## Project Structure
- `manuscript/`: Contains draft manuscript sections.
- `data/`: For aggregated epidemiological datasets.
- `literature/`: Summaries of key research papers.
